04 — Example

One call.
Full profile.

Enrich any domain and get back a complete verified record — contacts, tech stack, fit score, pitch angle.

Request # Enrich a company by domain
curl -X POST https://api.nulllimit.gg/enrich \
  -H "x-api-key: nl_your_key_here" \
  -H "Content-Type: application/json" \
  -d '{"url": "ugmonk.com", "type": "auto"}'

Response {
  "domain": "ugmonk.com",
  "name": "Ugmonk",
  "founder": { "name": "Jeff Sheldon", "email": "jeff@ugmonk.com" },
  "landed_cost_at_checkout": false,
  "checkout_verified": true,
  "zonos_fit_score": 8,
  "credits_used": 1
}
